# スマホ縦持ちの場合のゲーム表示デモ
スマホ縦持ちで画面が縦長の場合に、ゲーム画面を90度回転させて表示するデモです。


![h](https://github.com/user-attachments/assets/5c2bea9c-9191-4006-a6cf-310a92728415)
![v](https://github.com/user-attachments/assets/cbf03d48-8f09-4411-9cdb-efe55fc90f3d)


## 実行

yarn + viteで実行できます。

```
yarn && yarn dev
```

## ビルド

```
yarn build
````
